# CE-IA · Auditor de Trámites

Sistema gratuito y controlado desde WordPress para investigar y mantener actualizadas las páginas de trámites del Consejo de Estudiantes de la Universidad de Oviedo.

## Piloto seguro

- Proveedor exclusivo: Gemini en nivel gratuito.
- Sin OpenAI ni proveedores de pago alternativos.
- Sin publicación automática: toda propuesta requiere revisión humana en WordPress.
- Los workflows de auditoría y pruebas quedan en modo manual durante el piloto.
- La primera auditoría procesa como máximo un trámite.
- Las claves no se guardan en el repositorio.

El trabajador se conecta a la REST privada del plugin mediante el usuario técnico `ceia-worker` y una contraseña de aplicación de WordPress.
